# C# - Projeto Interativo

Projeto desenvolvido no ensino secundário com foco na aprendizagem de programação em C# e desenvolvimento de aplicações interativas.

## Tecnologias Utilizadas

* C#
* .NET
* Windows Forms

## Funcionalidades

### Jogo da Memória

* Sistema interativo de correspondência de cartas
* 3 níveis de dificuldade diferentes
* Lógica de pontuação e progressão

### Conversor de Temperatura

* Conversão entre diferentes unidades de temperatura
* Interface simples e interativa

## Objetivo do Projeto

Aplicar conceitos fundamentais de programação, lógica e desenvolvimento de interfaces gráficas através da criação de aplicações interativas em C#.

## Competências Desenvolvidas

* Desenvolvimento de interfaces gráficas
* Estruturas condicionais e ciclos
* Lógica de programação
* Organização de aplicações em C#

## Contexto

Projeto desenvolvido na disciplina de Aplicacões Informáticas durante o ensino secundário na área de Ciências e Tecnologias.
